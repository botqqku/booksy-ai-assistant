# AI Booking Assistant

A simple AI-powered booking assistant demo built with Node.js and a mock AI.  
This project demonstrates full-stack development skills, AI integration, and JSON data handling. Perfect for showcasing in your portfolio.

---

## Live Demo

Frontend demo (mock AI) is hosted on GitHub Pages:

[https://your_github_username.github.io/booksy-ai-assistant/](https://your_github_username.github.io/booksy-ai-assistant/)

> Note: The live demo uses a mock AI for demonstration purposes, no real OpenAI API key is required.

---

## Features

- Send a booking request via a frontend interface.
- Mock AI parses messages into structured JSON `{ service, date, time }`.
- Bookings are saved into `bookings.json`.
- Fully remote-friendly backend and frontend separation.
- Portfolio-ready, clean repository without secrets.
